    Overview

    Field sets are used to group the fields of an object and allow flexibility for fields displayed on the user interface. Dynamically, we can add, remove, and reorder the fields to display on Lightning Web Components (LWCs), pages, and sections throughout the application. Descriptions of LWC and field set locations are based on TMS defaults.

    Custom fields created on the objects listed in the tables below may also be added to field sets for that object.

    Mode-Specific means a field set can be overridden by Mode specific field sets (e.g., “TMS1 Truckload”). The following tables summarize field set usage in the TMS.
